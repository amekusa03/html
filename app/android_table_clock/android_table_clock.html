Android Table Clock

Android端末にインストールできる置き時計アプリです。

About this Project

古いスマホの再利用を検討しました。

Video

Development Environment

  • Kotlin
  • Android SDK

Key Features

  • シンプルなデザイン: 毎日使用するためのシンプルなデザインを提供します。
  • 高精度な時刻表示: 精度の高い時刻表示機能で、正確な時間を確認できます。
  • Android6.0以上対応: Android6.0以上のお使いの端末に対応しています。
