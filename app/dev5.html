Video

About this Project

Qtプロジェクトを簡単に起動するためのツールです。プロジェクトのビルドや実行をワンクリックで行えるようにします。

Development Environment

  • Qt
  • C++
  • QML

Key Features

  • ビルド自動化: ワンクリックでQtプロジェクトのビルドから実行までを完結。
  • 環境切り替え: 複数のQtバージョンやビルド設定を効率的に管理。
  • QMLプレビュー: UI開発をスムーズにするための高速な起動プロセス。
