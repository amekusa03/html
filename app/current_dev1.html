暗号化ブラウズラッパー

About this Project

画像やテキストファイルなどを暗号化し、ファイル自体のセキュリティを強化します。

このプロジェクトでは、管理者によって暗号化されたファイルを、 承認された閲覧者が安全にブラウザ上で復号・閲覧するためのラッパーツールを提供します。 これにより、ファイルの安全な管理と閲覧が可能になります。

Development Environment

  • .NET 8
  • C#
  • Avalonia UI

Key Features

  • 暗号化ファイルの復号: 暗号化されたファイルを安全に復号し、ブラウザで表示。
  • ユーザーフレンドリー: 直感的なUIで簡単に操作可能。
