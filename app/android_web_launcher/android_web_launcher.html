Android Web Launcher

Androidのアプリのように振る舞わせる「らん茶」

About this Project

特定のWebページをAndroidの独立したアプリのように振る舞わせる「らん茶」アプリです。 ユーザーは最大5つのプリセット（ドロワー表示）と、無制限のホーム画面ショートカットを管理できます。 これにより、特定のWebページを簡単にアクセスできるようになります。
また、静岡生まれの「らん茶」という名前は、このランチャーはお茶と同じ静岡生まれです。

Video

Development Environment

  • Kotlin
  • Android SDK

Key Features

  • Webページの独立したアプリとしての表示: 特定のWebページをAndroidの独立したアプリのように表示。
  • ユーザーフレンドリー: 直感的なUIで簡単に操作可能。

Out of Scope

  • 設定された名前をドロワーに表示されることはできませんでした。
  • Nova Launcherでは変更できる事、AQUOS HOMEでは変更できない事は確認しました。
