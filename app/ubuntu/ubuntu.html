Ubuntu導入ガイド

Linuxのディストリビューションと機能停止したWindows活用

About this Project

古くなって動作が重くなったWindows PCを、軽量なLinuxディストリビューションであるUbuntuに入れ替えることで、現役の端末として再利用するプロジェクトです。 廃棄コストを抑え、環境に優しいコンピューティングを実現しましょう。

Video

OS / Environment

  • Ubuntu 22.04 LTS
  • Lightweight Desktop Environment
  • Linux Kernel

Key Features

  • 高速化: Windowsに比べてシステムリソースの消費を大幅に削減。
  • セキュリティ: サポートの切れたOSから最新のセキュリティアップデートが受けられる環境へ。
  • ソフトウェア互換性: Webブラウジングやドキュメント作成に必要なアプリのセットアップ。
