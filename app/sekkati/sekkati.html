メモツール

About this Project

せっかちさんのためのメモ帳。さっと出して、書いて、しまう。

  • ホットキーで即起動: OS のショートカット設定からいつでも呼び出し。
  • 自動保存: フォーカスを失うと内容を自動で保存して終了。
  • ファイル管理不要: タイムスタンプで自動命名されるため、名前を考える手間がありません。
  • 時系列参照: 「今日」「昨日」などの単位で過去のメモを素早く閲覧。
  • 常駐型: バックグラウンドで待機し、2重起動を防ぎます。

Key Features

  • 高速なワークフロー: 思考を妨げない最小限のステップでメモを残せます。
  • Avalonia UI採用: .NET 10ベースのクロスプラットフォーム対応により、軽快な操作感を実現。
