ESP32 world Clock for Wokwi

ESP32で世界時計をシミュレートする

About this Project

これはWokwiプロジェクトです。LCD1602ディスプレイに時刻と日付を表示する世界時計です。時刻はNTP経由で同期されます。

Video

Development Environment

  • ESP32
  • C
  • Wokwi

Key Features

  • NTP時刻同期: インターネット経由で正確な時刻を取得し、自動的に同期します。
  • LCD1602表示: 限られた表示領域の中で、日付と時刻を効率的にレイアウト。
  • Wokwiシミュレーション: ブラウザ上で完結する開発環境により、ハードウェアなしで動作確認が可能。
