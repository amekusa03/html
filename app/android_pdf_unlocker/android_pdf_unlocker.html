Android PDF Unlocker

パスワード付きのPDFを設定されたパスワードで解除します。

About this Project

パスワード付きのPDFを前もって設定されたパスワードで解除し、PDFアプリに渡す、ラッパーアプリです。 毎月送信される給与明細などのパスワード付きPDFを、パスワードを入力する手間を省きます。

2026年5月2日現在：アプリ内でPDF参照まで可能としました。 自己評価で申し訳ありませんが、 このアプリの導入により、非常に便利にPDFを閲覧できるようになりました。

Video

Development Environment

  • Kotlin
  • Android SDK

Key Features

  • パスワード事前登録: アプリ内にパスワードを登録しておくことで、PDFを開く都度の入力が不要になります。
  • 自動解除: パスワード付きPDFを受信した際、登録済みのパスワードで自動的にロックを解除します。
  • アプリ内PDFビュワー: 解除後のPDFをアプリ内で直接閲覧できます。

Known Issues / Out of Scope

  • PDFの参照はこのアプリ内でのみ行います。外部のPDFアプリへの受け渡しには対応していません。
