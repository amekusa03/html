HTML to Blogger

ローカルにあるHTMLファイルと画像を、解析処理してBloggerに投稿

About this Project

ローカルにあるHTMLファイルと画像を、自動的に処理してBloggerに投稿するためのデスクトップアプリケーションです。 HTMLのクリーニング、画像への透かし追加、キーワードや位置情報の付与、そしてBloggerへのアップロードを実行します。

Video

Development Environment

  • Python 3.x
  • BeautifulSoup4 (HTML Parsing)
  • Pillow (Image Processing)
  • Google API Client

Key Features

  • HTMLクレンジング: 不要なタグや属性を除去し、Bloggerに最適な形式へ変換。
  • 画像の一括処理: ローカル画像をアップロードし、自動で透かし（Watermark）を合成。
  • Blogger API連携: 認証から投稿、タグ付けまでを完全に自動化。
