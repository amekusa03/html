Smart PC Project

Matter over Wi-FiによるPC電源管理システムの統合

Video

About this Project

本プロジェクトは、Ubuntu PCにESP32-C6を統合し、最新のスマートホーム規格「Matter」に対応させる試みです。 PCの電源状態を他の家電と同様にシームレスに管理・操作できるようにすることを目指しています。

System Architecture

ESP32-C6とPCマザーボードの間には、フォトカプラを用いた絶縁回路を介在させ、電気的な安全性を確保しています。

系 ESP32-C6 GPIO PCマザーボード
出力系 GPIO (OUT) PWR SW ピン (フォトカプラ経由)
入力系 GPIO (IN) PWR LED ピン (状態監視)
  • Microcontroller: ESP32-C6 (Matter over Wi-Fi)
  • Host OS: Ubuntu (with ACPI integration)
  • Protocol: Matter (On/Off Light/Switch)
  • Controller: Google Nest Hub / Android Device

Key Features

  • リモート操作: 外出先や音声コマンドによるPCの電源ON/OFF。
  • 状態監視: PCが稼働中か停止中かをリアルタイムで確認可能。
  • 自動化連携: Google Home等の自動化設定による、定時起動や一括オフ。
  • 安全設計: フォトカプラによる回路絶縁と、物理ボタンの併用を維持。

Development Environment

  • ESP-IDF / Matter SDK
  • Google Nest Hub (第2世代)
