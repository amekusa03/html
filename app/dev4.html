ESP32 Racing Game for Wokwi

ESP32を使用したレーシングゲーム

About this Project

ESP32を使用したレーシングゲームです。Wokwiで動作させることを想定しています。

Video

Development Environment

  • ESP32
  • C++ (Arduino Core)
  • Wokwi

Key Features

  • ESP32制御: ESP32のハードウェアを制御し、レーシングゲームの動作を実現。
  • Wokwiシミュレーション: Wokwiでゲームをシミュレーションし、開発効率を向上。
  • ユーザーアクション: ユーザーの入力に応じてゲームの進行を制御。
