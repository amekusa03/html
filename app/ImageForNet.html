ImageForNet

Web投稿用画像加工ツールです。クロスプラットフォーム対応（Windows, macOS, Linux）で、Webへのアップロード前などに、画像のEXIF情報の削除や、透かし（ウォーターマーク）の追加を簡単に行うことができます。

✨ 主な機能

  • EXIF情報の削除: プライバシー保護のために撮影情報を除去します。
  • 透かし（ウォーターマーク）の追加:
    • テキストの設定
    • 位置の調整（四隅）
    • 色、フォントサイズ、不透明度の設定
    • システムフォント（日本語対応）の自動選択

📥 インストール

Windows

  1. .NET Desktop Runtime 8.0 をダウンロードしてインストールします。
  2. アプリをダウンロードして解凍します。
  3. ImageForNet.exeを実行します。

macOS

  1. .NETをインストールします: 
    brew install dotnet
  2. アプリをダウンロードして解凍します。
  3. ImageForNetを実行します。

Linux (Ubuntu/Debian)

  1. .NETをインストールします: 
    sudo apt-get update && sudo apt-get install dotnet-runtime-8.0
  2. アプリをダウンロードして解凍します。
  3. ImageForNetを実行します。

🛠️ 開発者向け情報

使用技術

  • Framework: Avalonia UI (.NET 8)
  • Image Processing: SixLabors.ImageSharp
  • MVVM: CommunityToolkit.Mvvm

セットアップと実行

リポジトリをクローンし、依存関係を復元して実行します。

git clone https://github.com/amekusa03/ImageForNet.git
cd ImageForNet
dotnet restore
dotnet run --project ImageForNet
