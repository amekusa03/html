Avalonia学習

Avalonia UIフレームワークとMVVMパターンを学習

About this Project

これは、Avalonia UIフレームワークとMVVMパターンを学習するために作成した サンプルアプリケーションです。

Video

Development Environment

  • .NET 8.0 / C#
  • Avalonia UI v11
  • CommunityToolkit.Mvvm

Key Features

  • 軽量: Avalonia UIフレームワークを使用して、軽量で高速なアプリケーションを構築。
  • クロスプラットフォーム: Windows、Linux、macOSで動作するクロスプラットフォーム対応。
  • MVVMパターン: Model-View-ViewModelアーキテクチャを採用し、コードの分離と保守性を向上。
