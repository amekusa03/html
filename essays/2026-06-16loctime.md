# LocTime

2026/6/16 / kotlin

会社にいる時に、お昼の時間に気づかず、遅れそうになることがあったので、会社にいる時、お昼の時間に通知をしてくれるアプリを作った。

## About this Project

特定の場所と時間に合わせて通知する「LocTime」アプリ。

職場での休憩時間や、就業時間を数分前に通知するなど、ユーザーの生活スタイルに合わせて柔軟に通知を設定できる。
自宅にいる時は不要だけど、職場にいる時は時刻通知が欲しい、というようなシーンを想定している。
特定の場所と時間に合わせて時刻通知を受け取ることができる。 「LocTime」という名前は、様々な場所と時間に合わせた通知を可能にするという意味を持っている。

## Key Features

- **場所と時間の通知: **特定の場所と時間に合わせて通知を表示。
- **住所より場所を指定: **住所を入力することで、より正確な場所を指定できる。
- **ユーザーフレンドリー: **直感的なUIで簡単に操作できる。
