# Android PDF Unlocker

2026/03/12 / kotlin

毎月届く給与明細のパスワード入力が面倒だったので、androidアプリでパスワード付きPDFを設定されたパスワードで解除し、PDFアプリに渡す、ラッパーアプリを作った。

## About this Project

androidアプリでパスワード付きPDFを設定されたパスワードで解除し、PDFアプリに渡す、ラッパーアプリ。
毎月送信される給与明細などのパスワード付きPDFを、パスワードを入力する手間を省く。

2026年5月2日現在：アプリ内でPDF参照まで可能。
このアプリの導入により、非常に便利にPDFを閲覧できるようになりました。

## Key Features

- **パスワード事前登録: **アプリ内にパスワードを登録しておくことで、PDFを開く都度の入力が不要になる。
- **自動解除: **パスワード付きPDFを受信した際、登録済みのパスワードで自動的にロックを解除する。
- **アプリ内PDFビュワー: **解除後のPDFをアプリ内で直接閲覧できる。

## Out of Scope

- PDFの参照はこのアプリ内でのみ行います。外部のPDFアプリへの受け渡しには対応していません。
