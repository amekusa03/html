# HTML to Blogger

2026/01/30 / Python / Desktop Application

## 概要

ローカルにあるHTMLファイルと画像を、自動的に処理してBloggerに投稿するためのデスクトップアプリケーションです。
HTMLのクリーニング、画像への透かし追加、キーワードや位置情報の付与、そしてBloggerへのアップロードを実行します。

## Key Features

- **HTMLクレンジング: **不要なタグや属性を除去し、Bloggerに最適な形式へ変換。 
- **画像の一括処理: **自動で透かし（Watermark）を合成。 
- **画像のアップロード: **アップロードする画像を表示し手動アップロード。 
- **Blogger API連携: **認証から投稿、タグ付けまでを完全に自動化。
