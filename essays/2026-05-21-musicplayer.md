# Android Auto Music Player

2026/05/21 / kotlin

古いスマホを音楽プレイヤーとして再利用したいと思い、androidアプリを開発した。
ついでに車の中でも使えるようにAndroid Autoにも対応させた。

## About this Project

フォルダを指定してMP3ファイルを再生でき、Android Autoにも対応した。

設計・コード生成・デバッグまで、ほぼすべての工程を Claude Code との対話で進めました。
Android Autoのアートワーク表示やシャッフルボタンの不具合など、深い原因調査と修正もAIが主導して対応した。
生成されたコードは人間がレビューし、実機で動作確認を行っている。

## Key Features

- **フォルダ選択再生:** フォルダを指定してMP3ファイルを一括再生できる。
- **アルバムアート表示:** 曲に埋め込まれたアルバムアートを画面に表示できる。
- **シャッフル再生:** ランダムな順序で楽曲を再生できる。
- **検索機能:** 曲名・アーティスト名・アルバム名で絞り込み検索ができる。
- **バックグラウンド再生:** アプリを閉じてもフォアグラウンドサービスで再生を継続できる。
- **Android Auto 対応:** 車載ディスプレイから曲の操作ができる。
- **スキャン結果のキャッシュ:** 起動を素早く保ち、フォルダ変更時に自動で更新できる。

## Known Issues / Out of Scope

- **Googleアシスタントの音声操作（「〇〇をかけて」）:** インテントフィルタの追加と受け取り処理は実装済みだが、アシスタントがどのアプリに再生を委譲するかはOS側の判断に依存する。
YouTube MusicやSpotifyが優先されることがあり、本アプリへの誘導を保証できないため、実質的な検証ができていない。
