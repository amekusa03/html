# Image for net v3

2026/06/23 / C# / Avalonia UI / Windows,Linux,macOS

.NetでマルチOSを学習するために作成。
またネットに画像をアップする時に実用的であるように作成する。

## About this Project

**ver2.0**は、プレビュー画像を追加した。 **ver3.0**は、一括編集機能を追加した。 
**EXIF情報の削除:**プライバシー保護のために撮影情報を除去する。 **透かし（ウォーターマーク）の追加:**

* テキスト、位置（四隅）の調整
* 色、フォントサイズ、不透明度の設定
* システムフォント（日本語対応）の自動選択
**Avalonia UI採用:**.NET 8ベースのクロスプラットフォーム対応により、Windows、Linux、macOSで軽快な操作感を実現する。

## Key Features

* **個人情報保護:**画像に含まれる個人情報（EXIFデータ）を一括で除去。
* **日本語対応の透かし:**システムフォントを自動認識し、不透明度や色を自由に調整した透かしを合成。
* **Avalonia UI採用:**.NET 8ベースのクロスプラットフォーム対応により、軽快な操作感を実現。
