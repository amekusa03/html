# My PC Audio

2026/04/05

PC音声の変遷

## About this Journey

PCでの音楽鑑賞体験を向上させるため、10年以上にわたり試行錯誤してきたオーディオ環境の変遷の記録。
アナログ接続からデジタル出力への移行、スピーカーの自作やリプレイスなど、音の解像度とノイズ対策を追求したプロセス。

実質工数:8時間程度。 費用 ≈ ¥4,100 内訳 : YAMAHA ~~TSS-1: ¥30,000~~ TSS-20: ¥1,100 : YAMAHA ~~NS-C210~~ NS-C110: ¥3,000 : ~~DigiFi No.7: ¥3,000 : Pioneer TS-1019: ¥0 (廃車からスピーカーを流用) : ケーブル類: ¥0 (手持ちのものを使用) ~~

## Initial

PC付属のスピーカーを使用。音質は悪いが、手軽に音を出せる。

## Step 1

### System Components

- YAMAHA TSS-1

簡易ホームシアターサウンドシステムを購入。音質が向上し、PCでの音楽鑑賞が快適に。
ドルビープロロジックにて、2chの音声を5.1chに変換して出力する機能を搭載。
（アナログI／F）

## Step 2

### System Components

- YAMAHA TSS-1
- YAMAHA NS-C210

センタースピーカーを変更。
音がクリアになり、ボーカルが際立つように。センタースピーカーの重要性を実感。

## Step 3

#### System Components

- YAMAHA TSS-1 (スピーカー２本のみ使用)
- DigiFi No.7付録：USB-DAC + デジタルアンプ

デジタル化。 **DigiFi No.7付録 USB-DAC + デジタルアンプ**を採用。 アンプ変更に伴い、TSS-1のHiFiスピーカーのみとする。 解像度上昇。ボリーム操作が過敏。

## Step 4

#### System Components

- YAMAHA TSS-1
- Pioneer TS-1019

スピーカー交換。 **Pioneer TS-1019**(定格出力20W) -
                    エンクロージャーはダンボールで自作であり低音スカスカ。

## Step 5

#### System Components

- YAMAHA TSS-1
- YAMAHA NS-C210

Phase 2の状態に戻す。 最初のPhase 1から10年以上経過。
                    グラウンドループノイズらしきものが発生。

## Step 6

### System Components

- YAMAHA TSS-1
- YAMAHA NS-C210
- GIGABYTE GA-H87-HD3 - 製品情報

マザーボードから直接デジタル信号を出力。 **GIGABYTE GA-H87-HD3**-
Realtek ALC892のS/PDIF出力をTSS-1の同軸デジタル入力に接続。
デジタルケーブルは手作り、ノイズが減少。解像度上昇。低音までしっかり再現。

### Command memo for Linux

- alsamixer -
ALC892のS/PDIF出力を確認するコマンド

## Step 6.2

PC電源からアンプへ給電。 **TSS-1**
ACアダプタが故障したため、PC電源から給電するように改造。 本来、TSS-1の電源はDC15Vですが、これだけの為に電源を用意するのは常用としてはオーバースペックです。
苦肉の策でPC電源のPCIe電源コネクタから給電するように改造しました。 これにより、PCの電源を入れると同時にアンプも起動するようになり、利便性が向上しました。 ただし、PC電源の12Vラインから給電しているため、TSS-1の定格電圧よりも低い電圧が供給されることになります。これがアンプにどのような影響を与えるかは不明ですが、現状では問題なく動作しています。 電源の改造は自己責任で行っており、誤った接続や過電圧による機器の損傷のリスクがあるため、注意が必要です。

## Step 7

### System Components

- YAMAHA TSS-20
- YAMAHA NS-C110
- GIGABYTE GA-H87-HD3

TSS-1が不調により、アンプをTSS-20に変更（ヤフオクで1,100円）
センタースピーカーをTVに譲り、NS-C110に変更（ヤフオクで3,000円）
想定外だか音の広がりが増し、ボーカルもくっきり定位感も増した。

#### 課題

- アンプ、スピーカーサイズが大きい
- PCが買い替えとなったとき、デジタル出力がない可能性が大
