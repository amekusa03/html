# スピーカーのカタログスペックの読み方

スピーカーのカタログや製品ページに載っている数値・用語を、実際の選び方に活かすための整理。

## 1. 基本スペック項目

### 型式(方式)
- **2ウェイ / 3ウェイ**:低音〜高音を何個のユニットで分担するか。ウェイ数が多いほど各帯域を専用ユニットで鳴らせるが、価格・サイズも上がる
- **ブックシェルフ型 / トールボーイ型**:設置スタイルの分類。ブックシェルフは棚やスタンドに置く小型、トールボーイは床置きの縦長タイプ

### 再生周波数帯域(f特)
- 例:「45Hz〜60kHz」のように表記される
- **下限の数値が小さいほど低音の伸びが良い**目安になる(ただし出力レベルの記載がない場合は参考程度)
- 上限は人間の可聴域(〜20kHz)を超えることが多く、実用上の差は小さい
- カタログ値は「-10dB」など基準が製品ごとに違うことがあるため、他機種との横並び比較には注意

### インピーダンス(公称インピーダンス)
- 単位はΩ(オーム)。多くは **4Ω** か **6Ω / 8Ω**
- アンプ側の対応インピーダンスと合わせる必要がある。アンプの最低対応インピーダンスより低い数値のスピーカーを繋ぐと、アンプに負荷がかかりすぎる場合がある
- 数値が低いほど「駆動しやすいがアンプへの負荷が大きい」傾向

### 出力音圧レベル(能率・感度)
- 単位は **dB/W(1m)**。「1Wの入力を1m地点で測ったときの音圧」
- 数値が大きいほど**少ないアンプ出力で大きな音が出る=能率が良い**
- 目安:88dB以下=低能率、89〜91dB=標準、92dB以上=高能率
- 低能率のスピーカーは、同じ音量を出すのにアンプにより大きなパワーが必要

### 最大入力・定格入力
- **定格入力**:連続的に入力しても安全な目安の値(W)
- **最大入力(瞬間最大)**:一瞬だけ耐えられる値。カタログでは大きい数字が目立つが、実用上は定格入力の方が参考になる
- アンプの出力がスピーカーの定格入力を大きく超えないか確認する

### クロスオーバー周波数
- マルチウェイ機で、低音用と高音用のユニットを切り替える境界の周波数
- 例:「3kHz」なら、それより低い帯域をウーファー、高い帯域をツイーターが担当

### ユニット構成(口径)
- ウーファー(低音)、ミッドレンジ(中音)、ツイーター(高音)それぞれの振動板の口径(cm/mm)
- 一般に口径が大きいほど低音の量感は出やすいが、設置スペースとのトレードオフ

## 2. 選定時に見るべき優先順位の目安

1. **インピーダンスがアンプと適合するか**(必須条件)
2. **出力音圧レベル**とアンプの出力(W)のバランス — アンプがそこそこの出力なら低能率でも駆動できる
3. **再生周波数帯域**の下限 — 低音重視なら重要、会話メインの用途(映画のセリフなど)なら優先度は下がる
4. **サイズ・設置方式** — 設置場所に物理的に収まるか

## 3. 注意点

- カタログスペックは**測定条件が非公開・非統一**なことが多く、同じ数値でも実際の聴感が一致するとは限らない
- 数値だけで比較するより、**用途(音楽メインか映画メインか)に合わせて優先項目を絞る**のが実用的
- 同一メーカー・同一シリーズ内での比較は測定条件が揃っていることが多く、参考にしやすい
