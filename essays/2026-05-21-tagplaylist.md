# TagPlaylist Generator

2026/05/21 / Python / GUI / CLI

MP3のタグ情報からジャンル別プレイリストを自動生成する。

音楽ライブラリの整理を効率化するために開発したツールです。大量のMP3ファイルからID3タグ（ジャンル）をスキャンし、ジャンルごとに `.m3u `プレイリストを自動的に作成します。GUIとCLIの両方に対応しており、ポータビリティを重視して相対パスでプレイリストを書き出します。

## Key Features

* **自動ジャンル分類:** ID3v2タグのTCONフレームを元に分類する。 
* **相対パス生成:** プレイリストからの相対パスで記述するため、フォルダごと移動しても壊れない。 
* **GUI/CLI両対応:** ドラッグ＆ドロップ感覚で使えるGUIと、自動化に便利なCLIを備えている。 
* **再帰的スキャン:** 指定フォルダ以下のサブフォルダもすべて検索する。
