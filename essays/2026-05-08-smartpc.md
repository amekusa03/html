# [Canceled] Smart PC Project

2026-05-08 / C# / Matter over Wi-Fi 

Matter over Wi-FiによるPC電源管理システムの統合

## About this Project

本プロジェクトは、Ubuntu PCにESP32-C6を統合し、最新のスマートホーム規格「 **Matter **」に対応させる試みです。
PCの電源状態を他の家電と同様にシームレスに管理・操作できるようにすることを目指しています。

本プロジェクトはスキル不足により、中止にしました。

事由
・LCD付きESP32を購入したが、空きポートを間違え散々検証した結果、フォトカプラの抵抗を疑り、何度もハンダ付けした結果、
　フォトカプラを破壊してしまった。（パターン剥がれ）
・安全機能のつもりで状態が不整合となった場合は、OFFするように作成したところ、
　スイッチを連打し、PCの起動、シャットダウンを繰り返してしまった。
・PCのM/Bにあるpower LED、power SWの仕様が分からなかったが、実測して合わせようとして失敗してしまった。（明確に分からなかった）
