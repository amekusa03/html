# Lunch Memo

2026/07/01 / kotlin

ランチ予定を忘れる事が多く、管理するAndroidアプリを作った。

## About this Project

Lunch Memo は、日々のランチの予定や予約を管理するAndroidアプリ。
カレンダー形式のカード操作でメモをすばやく残せるように工夫し、
お昼の通知によって今日の予定をお知らせする。

## Key Features

- **スマートな記録:** カレンダー形式のカードをスワイプして、ランチ内容をすばやくメモできる。
- **お昼の通知:** 毎日12:00に、その日のランチメモを通知する。
- **モダンなデザイン:** Material 3 をフル活用し、滑らかなカードアニメーションとグラデーション背景を採用している。
- **通知権限対応:** Android 13 以上の通知権限に対応している。
- **クリーンなデータ保持:** 過去のメモを自動整理し、アプリを常に軽快に保つ。

## Setup

1. Android Studio (Ladybug 以降推奨) でプロジェクトを開く。
2. Gradle Sync を実行する。
3. Android デバイスまたはエミュレータでアプリを実行する。
