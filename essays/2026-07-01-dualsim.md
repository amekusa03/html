## 調べた

`SubscriptionManager`でアクティブなSIM一覧を取得して、それぞれの`subscriberId`でクエリをかければ、SIMごとの使用量が取れる。構造としてはシンプルだ。

```kotlin
val subs = subscriptionManager.activeSubscriptionInfoList
for (sub in subs) {
    val tm = telephonyManager.createForSubscriptionId(sub.subscriptionId)
    val id = tm.subscriberId
    // NetworkStatsManager でクエリ
}
```
