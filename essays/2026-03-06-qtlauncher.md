# Qt Project Launcher

2026/03/08 / Qt / C++ / QML

Qtプロジェクトを起動する時、QTを起動しプロジェクトファイルを選択するのにストレスを感じ。
プロジェクトのビルドや実行をワンクリックで行えるようにした。

## Key Features

- **ビルド自動化: **ワンクリックでQtプロジェクトのビルドから実行までを完結。 
- **環境切り替え: **複数のQtバージョンやビルド設定を効率的に管理。 
- **QMLプレビュー: **UI開発をスムーズにするための高速な起動プロセス。
