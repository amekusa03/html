# EncBrowse

2026/04/19 / Avalonia UI / C #

個人の安価なクラウドWebサーバーではアクセス権を設定する事が困難。
そのため画像やテキストファイルなどを暗号化し、ファイル自体のセキュリティを強化する。
それを設定、管理し、復号して閲覧するためのアプリケーション。

## About this Project

このアプリケーションでは、暗号化されたファイルを承認された閲覧者が安全に復号し、
ブラウザーのラッパーとして閲覧するためのアプリケーションを提供する。
これにより、ファイルの安全な管理と閲覧が可能になる。
