# Kicadのすすめ

2026.07.16　/ Kicad

## 概要

Libreoffice Draw で電子回路の図面を書いていましたが、
思うように書けず、Kicodを使ってみました。

## インストール

### アプリセンター

オススメ
アプリセンターを起動して「Kicad」と入力するだけで、インストールが完了します。
![アプリセンターのkicad](essays/2026-07-16-kicad.png)
### webサイト

https://www.kicad.org/

難しい
webサイトに行くと実行ファイルがありますが、
これでインストールするのは手がかかります。

### コマンドライン

私はコレです。
```bash
sudo apt update
sudo apt install --install-recommends kicad
```

## 使い勝手

### 日本語
全部しっかりとした日本語です。
コレなら細かな設定も安心。

### 出だし
よくある事ですが、どこから手を付けたら良いのか迷いますよね、
コレならアイコンをポチッとしてポチっと置くだけで最初の部品が置けます。
迷ったらメニューの「配置」から欲しい部品を選択しましょう。

### 配線
私は回路図しか使ってませんが、部品がたくさんあってびっくり。
配置を少し気をつければ、回路図はあっという間に書けました。

### バーション管理
Gitにも対応してます。
別ツールを使う必要がないし、更新監視もあるようです。

## 使ってみよう

あのファイルは使える？レイアウトは？とか気になるなら先ずは使ってみましょう。
タダだし。電気屋さんは結構使えるんじゃないかなぁ。
ちなみにPCBを3Dで見えたりもしますので、高さを揃えたいエンジニアには良さそうです。
