# Android Web Launcher

2026/04/02 / kotlin

Amazon StoreやYahooショッピング、ヤフオク、などのWebページで十分事足りるページを簡単に閲覧できるように
Androidのアプリのように振る舞わせる「らん茶」を作った。

## About this Project

特定のWebページをAndroidの独立したアプリのように振る舞わせるアプリ「らん茶」。
最大5つのプリセット（ドロワー表示）と、無制限のホーム画面ショートカットを管理できる。
これにより、特定のWebページを簡単にアクセスできるようになる。
「らん茶」という名前は、このランチャーが、お茶の名産地である静岡で生まれたことに由来している。

## Key Features

- **Webページの独立したアプリとしての表示: **特定のWebページをAndroidの独立したアプリのように表示。 
- **ユーザーフレンドリー: **直感的なUIで簡単に操作可能。

## Out of Scope

- ドロワーに任意の名前を表示させる機能は、現状未実装です。 
- ランチャーアプリ依存の挙動があり、Nova Launcherでは変更可能、AQUOS HOMEでは変更不可であることを確認しています。
