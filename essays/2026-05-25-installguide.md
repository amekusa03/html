# Androidアプリのインストール方法

2026/05/25 / kotlin

GitHubのソースコードをスマホで動かすまでの手順

## はじめに

GitHubで公開されているAndroidアプリを、自分のスマホに入れて使う方法をまとめる。
アプリ開発の経験がなくても、手順通りに進めばAndroid Studioを使ってスマホにインストールすることができる。
本ガイドでは、環境構築から実機への転送までを分かりやすく解説する。

## ステップ1：Android Studioのインストール

まずは開発ツールである「Android Studio」をパソコンに導入する。

1. [Android Studio公式サイト](https://developer.android.com/studio)へアクセスする。 
2. 「Download Android Studio」ボタンをクリックし、規約に同意してインストーラーをダウンロードする。 
3. ダウンロードしたファイルを実行し、基本的にはすべて「Next」のまま進めてインストールを完了させる。 
4. 初回起動時にいくつかのコンポーネントがダウンロードされる。少し時間がかかるので待つ。

## ステップ2：GitHubからコードを取得する

次に、動かしたいアプリのプログラム（ソースコード）をダウンロードする。 

1. GitHubのプロジェクトページ（例： [WebLauncher2](https://github.com/amekusa03/WebLauncher2) ）を開く。 
2. 緑色の「 < > Code」ボタンをクリックし、「Download ZIP」を選択する。 
3. ダウンロードしたZIPファイルを、パソコンの好きな場所（デスクトップなど）に展開（解凍）する。

## ステップ3：Android Studioでプロジェクトを開く

1. Android Studioを起動し、「Open」を選択する。 
2. 先ほど展開したフォルダを選択して開きます（Androidのアイコンがついたフォルダが目印です）。 
3. 右下で「Gradle Sync」が始まります。必要なライブラリなどを自動で集めてくれるので、終わるまで（バーが消えるまで）数分待ちます。

## ステップ4：スマホ側の準備（開発者オプション）

スマホにアプリを送るには、スマホ側で「デバッグモード」を有効にする必要がある。 

1. スマホの「設定」→「デバイス情報」を開く。 
2. 「ビルド番号」を7回連続でタップする。「開発者になりました」と表示される。 
3. 「設定」→「システム」→「開発者向けオプション」を開き、 **「USBデバッグ」をON **にする。 
4. スマホとパソコンをUSBケーブルで接続する。「USBデバッグを許可しますか？」と出たらOKを押す。

## ステップ5：スマホへインストール

1. Android Studioの上部にあるデバイス選択欄に、自分のスマホの名前が表示されていることを確認する。 
2. その右隣にある **緑色の再生ボタン（Run 'app'） **をクリックする。 
3. プログラムのビルドが始まり、成功すると自動的にスマホでアプリが立ち上がります。 
**注意： **ビルド中にエラーが出る場合は、Android Studioのバージョンが古い、あるいはスマホのOSバージョンが対応していない可能性があります。

## まとめ

一度環境を作ってしまえば、他のGitHubプロジェクトも同じ手順で試すことができる。
自分でソースコードを眺めて、少し数字や色を変えてみるだけでも、アプリ開発の第一歩になる。ぜひ楽しんでみてください！
